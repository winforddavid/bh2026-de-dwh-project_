# Data Warehouse and Analytics Project

Welcome to my **Databricks Data Lakehouse Project** by **Winford David**.

This repository contains my **Data Lakehouse implementation** built on Databricks, including datasets, notebooks, SQL examples, etc. 
---


## 🏗️ Architecture

This project uses the 5 pillars well architect **Medallion Architecture**:

### 🥉 Bronze Layer
- Raw data ingestion  
- Schema inference and storage as Delta tables  

### 🥈 Silver Layer
- Data cleaning and standardization  
- Type casting and validation  

### 🥇 Gold Layer
- Dimensional Data Model (Business Transformation)
- Ready for BI and analysis  

---

## 🛠️ Technologies Used

- Databricks  
- Apache Spark  
- PySpark  
- Spark SQL  
- Delta Lake  
- Unity Catalog
